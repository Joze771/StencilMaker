# StencilMaker

## Purpose
The purpose of this program is to simplify and automate stencil generation for single-component integrated circuit packages. It provides a user-friendly desktop interface where you can input specific component dimensions and automatically generate a 3D-printable `.stl` file, making it immediately ready for 3D printing. 

## Disclaimer
This is my first Python project, I made it as a way to understand this language new to me, because my background is only C and ASM. So don't be too harsh on me, I would like to receive any advice, both for the code itself and the program functioning.

## Features
* Parametric 3D stencil generation for **DIP** and **QFP** configurations.
* Interactive GUI built with PySimpleGUI.
* Real-time input validation to prevent invalid geometries.
* Dynamic support for multiple measurement units.
